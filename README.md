# Python Web Scraping & Data Extraction Projects

## Overview
This repository contains multiple Python projects focused on web scraping, data extraction, and automation.  
The scripts collect data from websites, APIs, and databases, then store the extracted information in CSV files for further analysis.

These projects demonstrate practical experience with web scraping, automation, and data processing using Python.

---

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Selenium
- CSV Processing
- JSON Handling
- MongoDB
- WebDriver Manager

---

## Projects Included

### Government Jobs Scraper
Scrapes job listings from a government jobs website and collects information such as job title, location, department, salary, and closing date.

Output file:
government_job.csv

---

### JustWatch Movies Data Scraper
Fetches popular movies and TV show data from the JustWatch API and extracts information such as ratings, titles, and streaming services.

Output file:
just_watch.csv

---

### Hospitality Industry Contact Data Processor
Processes stored MongoDB data and extracts company and employee contact information related to the hospitality industry.

Output file:
job_hospitality.csv

---

### LinkedIn Company Data Scraper
Uses Selenium automation to collect company details from LinkedIn pages including company name, employee count, address, and website.

Output file:
linkedin_63k6.csv

---

### Email Verification Automation Tool
Automates the verification of email addresses by checking them on an email verification website and recording the results.

Output file:
emails_data11.csv

---

### Trustpilot Review Scraper
Scrapes company reviews from Trustpilot including reviewer details, ratings, and company information.

Output file:
trust_pilot.csv

---

## How to Run the Scripts

1. Install required Python libraries

pip install requests beautifulsoup4 selenium webdriver-manager pymongo

2. Run the Python script

python script_name.py

3. The extracted data will be saved automatically in CSV format.
